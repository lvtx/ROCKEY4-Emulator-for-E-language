# ROCKEY4 Emulator for E-language
史上最最最完美的易语言破解补丁（没有之一）
翻箱底找到的，这是2009年写的补丁了，大概是从易语言3.8版本开始一直到现在最新的5.93版本，个人断断续续修改和更新，都十几年了老东西了~现在发出来做个备份！
使用本补丁程序无需安装驱动，直接安装好易语言后把补丁程序复制进去即可使用。

#### 使用说明

1. 本补丁程序是使用 RadASM 3.x 纯汇编编写和编译出来的，原理是利用 olepro32.dll API 劫持 e.exe 来实现模拟 ROCKEY4 加密狗进行破解。
 ![输入图片说明](https://images.gitee.com/uploads/images/2021/0727/000555_86795493_1232593.png "001.png")
 ![输入图片说明](https://images.gitee.com/uploads/images/2021/0727/000603_7a3e6493_1232593.png "002.png")
 ![输入图片说明](https://images.gitee.com/uploads/images/2021/0727/000610_33af109a_1232593.png "003.png")
 ![输入图片说明](https://images.gitee.com/uploads/images/2021/0727/000617_f82b9095_1232593.png "004.png")

2. 使用本补丁程序需要注意的一点就是，olepro32.dll 需要复制易语言的主程序 e.exe 所在的同一目录内才会有效。
